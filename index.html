SSBM Break the Targets Stage Randomizer

* Randomizes target positions, with stage-specific exclusion rules to prevent collisions
* Inserts new positions into a Gecko code template, which can then be used with Dolphin or Wii Homebrew


   

Because gecko codes are limited to a line length, the all stages code has been separated into two 13-stage codes.
(Dr. Mario through Ice Climbers // Kirby through Sheik (Seak))

Please enable one at a time for the codes to work.

Known issues:
* The two moving targets in Mewtwo's stage aren't affected
* Targets intentionally can generate partially inside walls
